## ⚡ 快速开始（3步）

### 1️⃣ 安装依赖

```bash
npm install
```

### 2️⃣ 配置数据库

创建 `.env` 文件：

```env
DATABASE_URL="postgresql://user:password@localhost:5432/medexam"
```

推送数据库架构并插入种子数据：

```bash
npm run db:push
npx prisma db seed
```

### 3️⃣ 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 🎉

## 🆘 常见问题

### Q: 数据库连接失败？
A: 检查 `.env` 文件中的 `DATABASE_URL` 是否正确配置

### Q: 种子数据如何插入？
A: 运行 `npx prisma db seed`

### Q: 如何重置数据库？
A: 运行 `npx prisma db push --force-reset` 然后 `npx prisma db seed`

### Q: 如何查看数据库？
A: 运行 `npx prisma studio` 打开可视化管理界面
