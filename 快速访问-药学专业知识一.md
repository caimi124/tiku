## 💾 数据库信息

### 连接信息
- **数据库**: PostgreSQL (Supabase)
- **表名**: questions
- **项目**: tiku2 (tparjdkxxtnentsdazfw)

### 查询示例
```sql
-- 查看所有药学专业知识（一）题目
SELECT id, content, question_type, chapter 
FROM questions 
WHERE subject = '药学专业知识（一）' 
  AND source_year = 2024 
ORDER BY chapter;

-- 查看包含图片的题目
SELECT id, content, ai_explanation 
FROM questions 
WHERE subject = '药学专业知识（一）' 
  AND ai_explanation IS NOT NULL;

-- 统计题型分布
SELECT chapter, COUNT(*) as count 
FROM questions 
WHERE subject = '药学专业知识（一）' 
GROUP BY chapter;
```
