## 🗄️ 四、数据库设计

### 4.1 核心数据表

#### 用户表 (users)
```sql
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  email VARCHAR(255) UNIQUE NOT NULL,
  username VARCHAR(100),
  password_hash VARCHAR(255),
  avatar_url TEXT,
  is_vip BOOLEAN DEFAULT FALSE,
  vip_expire_date TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

#### 题目表 (questions)
```sql
CREATE TABLE questions (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  exam_type VARCHAR(100) NOT NULL, -- 考试类型
  subject VARCHAR(100) NOT NULL,   -- 科目
  chapter VARCHAR(100),            -- 章节
  question_type VARCHAR(20) NOT NULL, -- 题型：单选/多选/判断
  content TEXT NOT NULL,           -- 题目内容
  options JSONB,                   -- 选项 [{key: 'A', value: '...'}]
  correct_answer TEXT NOT NULL,    -- 正确答案
  explanation TEXT,                -- 解析
  ai_explanation TEXT,             -- AI生成的解析
  difficulty INTEGER DEFAULT 1,    -- 难度：1-5
  knowledge_points TEXT[],         -- 知识点标签
  is_published BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

#### 用户答题记录 (user_answers)
```sql
CREATE TABLE user_answers (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID REFERENCES users(id),
  question_id UUID REFERENCES questions(id),
  user_answer TEXT,
  is_correct BOOLEAN,
  time_spent INTEGER,              -- 答题用时（秒）
  answered_at TIMESTAMP DEFAULT NOW()
);
```

#### 错题本 (wrong_questions)
```sql
CREATE TABLE wrong_questions (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID REFERENCES users(id),
  question_id UUID REFERENCES questions(id),
  wrong_count INTEGER DEFAULT 1,
  last_wrong_at TIMESTAMP DEFAULT NOW(),
  is_mastered BOOLEAN DEFAULT FALSE,
  UNIQUE(user_id, question_id)
);
```

#### 收藏表 (favorites)
```sql
CREATE TABLE favorites (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID REFERENCES users(id),
  question_id UUID REFERENCES questions(id),
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(user_id, question_id)
);
```

#### 学习记录 (study_sessions)
```sql
CREATE TABLE study_sessions (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID REFERENCES users(id),
  exam_type VARCHAR(100),
  questions_count INTEGER,
  correct_count INTEGER,
  time_spent INTEGER,
  started_at TIMESTAMP,
  ended_at TIMESTAMP
);
```

### 4.2 数据库关系图

  ├── user_answers (答题记录) ──→ questions (题目)
  ├── wrong_questions (错题本) ──→ questions

### 5.2 错题本智能分析

#### 功能特性
- 自动收集错题
- 统计错误次数
- 分析薄弱知识点
- 生成复习计划
- AI 个性化学习建议

#### 算法逻辑
```javascript
// 薄弱知识点分析算法
function analyzeWeakPoints(userAnswers) {
  const knowledgePointStats = {};
  
  userAnswers.forEach(answer => {
    if (!answer.is_correct) {
      answer.question.knowledge_points.forEach(point => {
        if (!knowledgePointStats[point]) {
          knowledgePointStats[point] = { total: 0, wrong: 0 };
        }
        knowledgePointStats[point].total++;
        knowledgePointStats[point].wrong++;
      });
    }
  });
  
  // 计算错误率，排序返回最薄弱的知识点
  return Object.entries(knowledgePointStats)
    .map(([point, stats]) => ({
      point,
      errorRate: stats.wrong / stats.total,
      wrongCount: stats.wrong
    }))
    .sort((a, b) => b.errorRate - a.errorRate)
    .slice(0, 10);
}
```
