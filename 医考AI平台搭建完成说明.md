## 🚀 部署指南

### 1. 环境变量配置

在 Vercel 或本地 `.env` 文件中配置：

```env
# 数据库（必需）
DATABASE_URL="your_postgresql_database_url"

# NextAuth（如需登录功能）
NEXTAUTH_SECRET="your_secret_key"
NEXTAUTH_URL="https://yikaobiguo.com"

# 其他可选配置
NODE_ENV="production"
```

### 2. 数据库初始化

```bash
# 推送数据库架构
npm run db:push

# 生成Prisma Client
npx prisma generate

# 插入种子数据（可选）
npx prisma db seed
```

### 3. 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 4. Vercel部署

1. 将代码推送到GitHub
2. 在Vercel中导入项目
3. 配置环境变量（DATABASE_URL等）
4. 部署
