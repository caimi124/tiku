## 🔧 技术实现

### 数据库设计
```sql
表名: questions
主键: id (UUID)
索引: 
  - source_year (年份索引)
  - question_type (题型索引)
  - subject (科目索引)
  - exam_type (考试类型索引)
```

### API接口
```typescript
GET /api/questions
参数:
  - sourceYear: 年份 (2022/2023/2024)
  - subject: 中药学综合知识与技能
  - limit: 返回数量 (默认200)
  
返回格式:
{
  success: true,
  data: {
    questions: [...],  // 自动转换为数组格式
    total: 120
  }
}
```

### 数据格式转换
```typescript
// 数据库存储（对象格式）
options: {
  A: "选项A内容",
  B: "选项B内容",
  ...
}

// API返回（数组格式，前端需要）
options: [
  { key: "A", value: "选项A内容" },
  { key: "B", value: "选项B内容" },
  ...
]
```

### 题型映射
```typescript
数据库字段 question_type:
- "最佳选择题" → 1-40题
- "配伍选择题" → 41-90题
- "综合分析题" → 91-110题
- "多项选择题" → 111-120题
```
