# 2024年执业药师中药学综合知识与技能真题导入说明

## 📋 文件说明

- `import-all-120-questions.ts` - 主导入脚本
- `完整120道题目数据.txt` - 包含所有120道题目的完整数据（请按格式复制到脚本中）

## 🚀 导入步骤

### 步骤1: 复制题目数据

打开 `完整120道题目数据.txt` 文件，复制所有题目数据。

### 步骤2: 粘贴到导入脚本

打开 `prisma/import-all-120-questions.ts` 文件，找到这一行：

```typescript
const allQuestions: QuestionData[] = [
  // TODO: 将下方示例题目替换为完整的120道题目
```

将复制的数据粘贴到数组中，替换示例题目。

### 步骤3: 运行导入脚本

在项目根目录执行：

```bash
npx tsx prisma/import-all-120-questions.ts
```

或者：

```bash
ts-node prisma/import-all-120-questions.ts
```

### 步骤4: 查看导入结果

脚本会显示详细的导入进度和统计信息：

```
✅ 成功导入: 120 道题目
❌ 导入失败: 0 道题目
📝 总计题目: 120 道
🎯 成功率: 100.00%
```

## 📝 题目数据格式说明

每道题目需包含以下字段：

```typescript
{
  examType: '执业药师',
  subject: '中药学综合知识与技能',
  chapter: '章节名称',
  questionType: 'single' | 'multiple' | 'case',  // single=单选, multiple=多选, case=案例
  content: '题目内容',
  options: [
    {key:'A', value:'选项A内容'},
    {key:'B', value:'选项B内容'},
    // ...
  ],
  correctAnswer: 'A',  // 或 'ABC' (多选题)
  explanation: '解题思路',
  difficulty: 1-3,  // 难度：1=简单, 2=中等, 3=困难
  knowledgePoints: ['知识点1', '知识点2'],
  sourceType: '历年真题',
  sourceYear: 2024
}
```

## ⚠️ 注意事项

1. 确保已配置好 `.env.local` 文件中的数据库连接
2. 运行前会清理2024年的旧数据，请注意备份
3. 如遇到导入失败，检查控制台错误信息

## 🔧 故障排除

### 问题：数据库连接失败
**解决**：检查 `.env.local` 中的 `DATABASE_URL` 配置

### 问题：Prisma客户端未生成
**解决**：运行 `npx prisma generate`

### 问题：题目数据格式错误
**解决**：检查JSON格式，确保所有字段都正确填写

## 📞 需要帮助？

如有问题，请检查：
1. 数据库连接是否正常
2. Prisma schema是否正确
3. 题目数据格式是否符合要求
