# 📘 历年真题按科目分类 - 使用指南

## 🎯 新功能说明

我们为您创建了全新的**按科目分类**的历年真题页面，让您可以更系统地学习执业药师考试各科目的真题。

---

## 📂 访问方式

### 方式1：使用新页面（推荐）

访问新的按科目分类页面：
```
https://yikaobiguo.com/practice/history-by-subject?exam=pharmacist
```

### 方式2：应用为默认页面

如果您想将按科目分类设为默认页面，需要在服务器上执行：

```bash
cd E:\tiku

# 备份原页面
git mv app/practice/history/page.tsx app/practice/history/page-by-year.tsx

# 使用按科目分类作为默认页面
git mv app/practice/history/page-by-subject.tsx app/practice/history/page.tsx

# 提交更改
git add .
git commit -m "设置按科目分类为历年真题默认页面"
git push
```

执行后，原URL `https://yikaobiguo.com/practice/history?exam=pharmacist` 将显示按科目分类的页面。

## 🎨 页面特性

### 1. **专业类别选择** 🏥

页面顶部提供两个专业类别卡片：

#### 药学类 💊
- 药学专业知识（一）
- 药学专业知识（二）

#### 中药学类 🌿
- 中药学专业知识（一）

- 药事管理与法规
- 中药学综合知识与技能

**特点：**
- 🎨 蓝色主题（药学类）/ 绿色主题（中药学类）
- ✅ 清晰的选中状态提示
- 📱 大按钮设计，移动端友好

---

### 2. **科目卡片展示** 📚

每个科目显示为独立的卡片，包含：

#### 科目信息
- 📖 科目完整名称
- 📊 总题数统计
- 📅 包含的年份数量
- ✅ 已完成题数（如果有）

#### 年份快速入口
三个年份按钮，点击直接进入：
- **2024年** - 最新真题 🔥
- **2023年** - 热门真题 ⭐
- **2022年** - 经典真题 📖

#### 操作按钮
- **开始练习** - 进入最新年份的题目
- **统计图标** - 查看详细统计（规划中）

---

### 3. **色彩编码系统** 🎨

不同科目使用不同颜色区分：
- 🔵 **蓝色** - 第1个科目
- 🟢 **绿色** - 第2个科目
- 🟣 **紫色** - 第3个科目
- 🟠 **橙色** - 第4个科目

**好处：**
- 快速识别不同科目
- 视觉层次清晰
- 提升用户体验

---

### 4. **学习建议** 💡

页面底部提供专业的学习建议：

1. **系统学习** - 先选择一个科目，完成所有年份
2. **从新到旧** - 从2024年开始，了解最新趋势
3. **对比总结** - 对比不同年份，总结高频考点
4. **重点科目** - 药事管理与法规是共同科目
5. **建议顺序** - 专业知识（一）→（二）→ 综合 → 法规

## 🎯 使用场景

### 场景1：新手入门
**推荐路径：**
1. 选择专业类别（药学或中药学）
2. 从第一个科目开始：专业知识（一）
3. 点击"2024年"进入最新真题
4. 完成后再学习2023年、2022年
5. 一个科目完成后，进入下一个科目

**好处：** 系统学习，扎实基础

---

### 场景2：冲刺阶段
**推荐路径：**
1. 查看所有科目的完成情况
2. 重点攻克薄弱科目
3. 快速浏览已掌握的科目
4. 做最新年份查漏补缺

**好处：** 有针对性，高效备考

---

### 场景3：专项突破
**推荐路径：**
1. 选择一个薄弱科目
2. 从最旧的年份开始（2022年）
3. 逐年推进到最新（2024年）
4. 总结该科目的考点变化趋势

**好处：** 深入理解，把握规律
