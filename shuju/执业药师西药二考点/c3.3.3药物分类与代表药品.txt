第一阶段 建立框架 初学

【考点 c3.3.3｜M02｜本页定位】
本页是平喘药的“总纲地图”，帮你三分钟理清所有平喘药的根本归属。记住两大战略目标：一是“立刻扩张支气管”（治标），二是“长期控制炎症”（治本）。所有药名都对号入座，你就赢了八成。

【考点 c3.3.3｜M03｜考什么 & 怎么考】
作为总论，考试几乎只考一件事：精准分类。给你一个药名或缩写（如SABA、ICS），让你准确归入其所属的类别。这是整个平喘药章节所有后续学习和做题的基石。

【考点 c3.3.3｜M04｜核心结构】
平喘药分类的唯一核心，就是看它的 “战略目标” 是什么：
第一战略：扩张支气管（快速治标，缓解症状）
这类药是“消防队”，目标是快速打开气道。

β2受体激动剂（打开气道的主力军）

短效SABA（急救队）：沙丁胺醇、特布他林。

长效LABA（长效驻防队）：福莫特罗、沙美特罗（名字带“特罗”）。

M胆碱受体拮抗剂（解除气道痉挛的副手）

短效SAMA：异丙托溴铵。

长效LAMA：噻托溴铵（名字带“托溴铵”）。

黄嘌呤类（磷酸二酯酶抑制剂）（老牌预备队）：茶碱、氨茶碱。

第二战略：抗炎/免疫调节（长期治本，控制病情）
这类药是“防洪指挥部”，目标是消除根本的炎症。

吸入型糖皮质激素ICS（抗炎核心部队）：布地奈德、氟替卡松、倍氯米松（名字带“松”或“奈德”）。

白三烯受体拮抗剂LTRA（精准干扰炎症信号）：孟鲁司特、普仑司特（名字带“司特”）。

其他特种部队：

抗IgE单抗（针对过敏）：奥马珠单抗。

肥大细胞膜稳定剂：色甘酸钠。

【考点 c3.3.3｜M05｜必背要点】
初学建立最牢固的分类框架，只背这“三组核心对应”：

急救双雄：短效SABA = 沙丁胺醇、特布他林（用于急性发作）。

抗炎双核：ICS = 布地奈德/氟替卡松（治本核心）；LTRA = 孟鲁司特（口服抗炎）。

长效双塔：长效抗胆碱LAMA = 噻托溴铵；长效β2激动剂LABA = XX特罗。

第二阶段 复习查漏 默认推荐

【考点 c3.3.3｜M03｜考什么 & 怎么考】
在总论部分，命题人所有心思都用在制造分类混淆上，尤其是利用缩写和药名字根：
陷阱一：“缩写-药名”对不上。这是最主要的坑。把“SABA”的选项里塞进“沙美特罗”（那是LABA），或把“LAMA”的选项里塞进“异丙托溴铵”（那是SAMA）。
陷阱二：“战略目标”混淆。把“治标”的支气管扩张剂（如沙丁胺醇）说成是“抗炎治本”的药，或者把“治本”的抗炎药（如布地奈德）说成能“快速缓解急性喘息”。
陷阱三：“词根暗示”张冠李戴。利用“特罗”、“托溴铵”、“司特”、“松”这些字根，故意把药放错类别。比如，把带“特罗”的药归到抗胆碱类。

【考点 c3.3.3｜M04｜核心结构】
总论的核心，就是提供一套万能的“分类检索系统”，遇到任何药名或缩写，都能快速定位：
检索第一步：看“战略目标”

如果题干或选项提到 “快速缓解”、“急性发作”、“按需使用” -> 去 “扩张支气管” 的阵营里找答案（SABA、SAMA）。

如果题干或选项提到 “长期控制”、“抗炎”、“预防发作”、“每日使用” -> 去 “抗炎/免疫调节” 的阵营里找答案（ICS、LTRA、LABA、LAMA）。
*注：LABA和LAMA虽属扩张剂，但因其长效，主要用于长期联合控制。
检索第二步：认“缩写标签”

看到 SABA -> 只对应 沙丁胺醇、特布他林。

看到 SAMA -> 只对应 异丙托溴铵。

看到 LAMA -> 首选 噻托溴铵。

看到 ICS -> 对应 布地奈德、氟替卡松、倍氯米松。

看到 LTRA -> 对应 孟鲁司特。
检索第三步：用“字根密码”验证

药名带 “特罗” （如福莫特罗）-> 极大概率是 LABA（β2激动剂）。

药名带 “托溴铵” -> 极大概率是 抗胆碱药（噻托溴铵等）。

药名带 “司特” -> 极大概率是 LTRA（白三烯受体拮抗剂）。

药名带 “松” 或 “奈德” -> 极大概率是 糖皮质激素（ICS）。

【考点 c3.3.3｜M05｜必背要点】
必须死记（这是分类题的“钢铁法则”）：
一，战略区分：沙丁胺醇（SABA）是急救的；布地奈德（ICS）是抗炎治本的。两者角色绝对不能互换。
二，缩写铁律：

SABA = 沙丁胺醇/特布他林。

SAMA = 异丙托溴铵（唯一答案）。

ICS = 布地奈德/氟替卡松。

LTRA = 孟鲁司特。
三，字根密码：“特罗”是LABA，“托溴”是抗胆碱，“司特”是LTRA，“松/奈德”是激素。这是快速验证的利器。
理解记忆（用于应对复杂描述）：
四，茶碱类自成一家（黄嘌呤类），不直接属于上述任何缩写体系，但属于“扩张支气管”战略。
五，LABA（如福莫特罗）和LAMA（如噻托溴铵） 虽为长效扩张剂，但不用于急性急救，而是与ICS联合用于长期控制。

【考点 c3.3.3｜M06｜解题逻辑与秒杀规则】
解题核心：先定战略，再对缩写，后用字根核验。
错误选项的经典长相（专门破坏你的分类体系）：

“短效β2受体激动剂（SABA）的代表药物是沙美特罗。”（错！ 沙美特罗是长效LABA）

“异丙托溴铵属于长效抗胆碱药（LAMA）。”（错！ 它是短效SAMA）

“孟鲁司特是一种吸入型糖皮质激素（ICS）。”（错！ 它是口服的LTRA）

“布地奈德可用于哮喘急性发作的快速缓解。”（错！ 它是抗炎药，起效慢，不能急救）
题干/选项关键词秒杀路径：

看到 “SABA” 或 “急性发作首选” -> 秒选 沙丁胺醇 或 特布他林。

看到 “SAMA” -> 秒选 异丙托溴铵。

看到 “ICS” 或 “抗炎控制基础用药” -> 秒选 布地奈德 或 氟替卡松。

看到 “LTRA” 或 “白三烯受体拮抗剂” -> 秒选 孟鲁司特。

看到药名 “XX特罗” -> 归类为 LABA（除非是特布他林，但特布他林是SABA，此为例外需单独记）。

看到题干要求 “快速缓解”，但选项是 布地奈德、孟鲁司特、茶碱 -> 优先选 茶碱（扩张剂），其余两者非急救首选。

第三阶段 冲刺秒杀 考前

【考点 c3.3.3｜M05｜必背要点】
最后十秒钟，只背这“四组生死对应”：

急救：SABA -> 沙丁胺醇。

抗炎核心：ICS -> 布地奈德。

口服抗炎：LTRA -> 孟鲁司特。

长短标识：SAMA -> 异丙托溴铵；LAMA -> 噻托溴铵。

【考点 c3.3.3｜M06｜解题逻辑与秒杀规则】
考场条件反射，看到就选：

看到 SABA/急性发作 -> 直接勾选 沙丁胺醇。

看到 ICS -> 直接勾选 布地奈德 或 氟替卡松。

看到 LTRA/白三烯 -> 直接勾选 孟鲁司特。

看到 SAMA -> 直接勾选 异丙托溴铵。

看到 LAMA -> 直接勾选 噻托溴铵。

看到选项把 沙美特罗 说成SABA，或把 异丙托溴铵 说成LAMA -> 直接判错。
