第一阶段 建立框架 初学

【考点 c13.4.2｜M02｜本页定位】
本页用于帮助考生在2分钟内，掌握外用糖皮质激素临床决策的核心逻辑：根据“病-人-药”三个维度，快速做出选药、用法的基本判断。

【考点 c13.4.2｜M03｜考什么 & 怎么考】
本考点是分类知识的临床应用。考题会结合一个具体的“病”（皮损类型/部位）和“人”（年龄、特殊状况），让你判断“药”的效价选择、主要风险或禁忌。这是典型的情景综合分析题。

【考点 c13.4.2｜M04｜核心结构】
临床评价的核心决策遵循以下铁三角模型：
一、 看“病”选强度（病情决定药力）

重度/肥厚性皮损（如银屑病、斑秃） → 需要强效/极强效敲门。

轻中度/薄嫩部位皮损（如面部、眼睑、婴儿湿疹） → 首选弱效或中效。
二、 看“人”定禁忌（人群决定红线）

儿童、老年人、孕妇 → 弱效优先，警惕系统吸收。

皮肤有感染、溃疡、萎缩 → 禁用或慎用。
三、 看“药”知风险（药力决定副作用）

强效/极强效 + 长期/大面积用 → 风险：激素依赖性皮炎、皮肤萎缩等。

任何效价，用于面部、会阴等薄嫩部位 → 风险均增大。

【考点 c13.4.2｜M05｜必背要点】
初学必须刻入脑海的三条“黄金法则”：

选药法则：治重度皮损（银屑病、斑秃）起始要用强效/极强效激素。

安全法则：给儿童、老年人或用在面部、眼睑，首选弱效激素。

风险法则：长期、大面积使用强效激素，最可能导致激素依赖性皮炎。

第二阶段 复习查漏 默认推荐

【考点 c13.4.2｜M03｜考什么 & 怎么考】
命题陷阱深度解析：

病-药错配陷阱：将“重度肥厚性斑块（如银屑病）”与“弱效激素”匹配，或将“面部轻度皮炎”与“强效激素”匹配，考查强度选择的精准性。

人-药禁忌陷阱：题干描述“儿童特应性皮炎”或“孕妇湿疹”，但选项中包含“强效含氟激素”，考查特殊人群的用药安全红线。

风险归因陷阱：描述“使用后出现皮肤变薄、毛细血管扩张”，要求反推最可能的原因（长期使用强效激素于面部），而非单纯识别该副作用。

【考点 c13.4.2｜M04｜核心结构】
外用糖皮质激素临床决策完整指南：
一、 效价选择决策树（根据“病”和“部位”）

首选强效/极强效（攻坚）：

适用情况：重度、肥厚、角化过度的皮损，如慢性斑块状银屑病、扁平苔藓、结节性痒疹、斑秃（局部）。

关键逻辑：需要强效抗炎力快速控制，但仅用于短期初始治疗，控制后需降阶梯。

选中效（主力）：

适用情况：特应性皮炎、湿疹（非面部）、神经性皮炎等中度炎症。

关键逻辑：平衡疗效与安全性，是许多慢性炎症性皮肤病的主力军。

选弱效（保底）：

适用情况：面部、眼睑、间擦部位（腋下、腹股沟）的皮炎；轻度红斑丘疹；婴幼儿皮炎；需长期或大面积使用时。

关键逻辑：薄嫩皮肤和高吸收部位，安全性第一。

二、 核心风险与不良反应（与“药”和“用法”强相关）

局部不良反应（常见）：

皮肤萎缩、毛细血管扩张、多毛 → 与强效激素、长期使用、薄嫩部位相关。

诱发/加重感染（细菌、真菌、病毒） → 单用激素于感染性皮损的典型恶果。

激素依赖性皮炎/玫瑰痤疮 → 长期外用中强效激素于面部的经典后果。

系统性不良反应（少见但严重）：

库欣综合征、下丘脑-垂体轴抑制 → 见于长期、大面积使用强效/极强效激素，尤其在儿童。

三、 绝对禁忌与重要注意事项（“人”与“病”的禁区）

绝对禁忌部位：皮肤溃疡、萎缩部位。

禁忌病症：未控制的细菌、真菌、病毒感染（如单纯疱疹、脓疱疮）。

特殊人群：

儿童/老年人：皮肤薄，吸收好，优先选择弱效。

孕妇（尤其孕早期）：避免使用含氟的强效激素（如氟轻松），因有潜在风险。

哺乳期妇女：避免在乳房部位使用。

【考点 c13.4.2｜M05｜必背要点】
必须死记（临床决策的基石）：

效价选择：重度肥厚皮损（银屑病等）→ 起始用强效/极强效。

人群选择：儿童、面部、眼睑 → 首选弱效。

典型风险：长期用于面部的强效激素 → 导致激素依赖性皮炎。
理解记忆（掌握逻辑）：

不良反应的严重程度与激素效价强度和使用时间/面积成正比。

“感染”是外用激素的常见干扰项，原则是“有感染，先抗感染，或复方制剂”。

【考点 c13.4.2｜M06｜解题逻辑与秒杀规则】
临床情景题三步分析法：
第1步：抓取题干中的“核心标签”。

标签“重度、肥厚、银屑病、斑秃” → 治疗方案中应包含【强效/极强效激素】。

标签“儿童、婴儿、面部、眼睑” → 治疗方案中应首选【弱效激素】。

标签“长期使用、面部红斑、停用后反弹” → 最可能诊断【激素依赖性皮炎】。
第2步：识别“禁忌红线”。

题干出现“皮肤感染、溃疡、萎缩” → 直接排除【单用激素】的选项，或选【禁用】。

题干强调“孕妇、哺乳期” → 警惕并排除【强效含氟激素】或【乳房部位使用】。
第3步：关联“风险归因”。

题干描述副作用“皮肤变薄、毛细血管扩张” → 原因必与【强效激素、长期使用、薄嫩部位】相关。

第三阶段 冲刺秒杀 考前

【考点 c13.4.2｜M05｜必背要点】
考前速记三大“决策开关”：

开关一（强攻）：银屑病/斑秃 → 强效/极强效。

开关二（维稳）：儿童/面部/眼睑 → 弱效。

开关三（警示）：长期用强效于面部 → 激素依赖性皮炎。

【考点 c13.4.2｜M06｜解题逻辑与秒杀规则】
考场秒杀，关键词直接触发结论：

银屑病 / 斑秃 / 肥厚皮损 → 选强效或极强效激素。

儿童 / 面部 / 眼睑皮炎 → 选弱效激素。

长期用激素后反跳 / 面部红血丝 → 诊断为激素依赖性皮炎。

皮肤感染 / 溃疡 / 萎缩 → 禁用或慎用激素。

孕妇 → 避免含氟强效激素。
