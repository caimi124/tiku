第一阶段 建立框架 初学

【考点 c2.2.1｜M02｜本页定位】
本页帮你5分钟理清抗风湿药的“家族族谱”，所有考点都围绕这个核心问题：哪些药是治标，哪些药是治本？

【考点 c2.2.1｜M03｜考什么 & 怎么考】
考试90%的题目都在考分类，就两个核心考法：

“对号入座”题：给你一个药名（尤其是那些“洋气”的生物制剂），问它属于哪一类。

“分清敌我”题：给你一句描述（比如“延缓关节破坏”），让你判断这是哪类药的特点。

【考点 c2.2.1｜M04｜核心结构】
记住这个“治标”与“治本”的分层结构，这是总纲：

治标药（控制症状，治不了根）

非甾体抗炎药（NSAIDs）：如布洛芬、双氯芬酸。

糖皮质激素：如泼尼松。

治本药（改善病情，延缓关节破坏）

传统合成类（核心基础）：如甲氨蝶呤、来氟米特。

生物制剂/靶向类（“精确制导导弹”）：如依那西普、托法替布。

【考点 c2.2.1｜M05｜必背要点】
现在只需死记这三条，构筑分类框架：

布洛芬、双氯芬酸是“治标”的NSAIDs，只能止痛消肿。

甲氨蝶呤是“治本”的传统改善病情药（DMARDs）的核心大哥。

依那西普、阿达木单抗、托法替布这些名字“花哨”的，都属于高级的“治本”靶向药。

第二阶段 复习查漏 默认推荐

【考点 c2.2.1｜M03｜考什么 & 怎么考】
命题陷阱高度集中在概念混淆，专考你记不牢：
陷阱一：“治标”“治本”傻傻分不清。把布洛芬说成能“延缓关节破坏”，或者把甲氨蝶呤说成“快速镇痛”。
陷阱二：“老药”“新药”张冠李戴。把生物制剂（如依那西普）划到传统合成药里，或者反过来。
陷阱三：“机制标签”贴错门。考题常出现TNF、IL、JAK这些靶点缩写，故意配错对应的代表药。
陷阱四：“用药顺序”乱排座次。忽略传统合成药是首选基础，一上来就用贵且风险高的生物制剂。

【考点 c2.2.1｜M04｜核心结构】
这是完整的“临床决策树”，做题就按这个思路走：
第一步：判断治疗目标 -> 选对大类

如果题干问“快速缓解关节肿痛” -> 答案在治标药里找：NSAIDs（布洛芬） 或短期用激素（泼尼松）。

如果题干问“控制病情发展、延缓关节破坏” -> 答案必须锁定在治本药（DMARDs）里。

第二步：确定大类后 -> 细化选择

如果确定用“治本药”，再看情况：

首选且最常用的是传统合成类，核心是甲氨蝶呤。

只有当传统药效果不好或病情重时，才考虑升级到生物制剂或靶向药。

第三步：识别具体药物 -> 看“身份证”

看到 “单抗”（如阿达木单抗）、“融合蛋白”（如依那西普） -> 这是生物制剂，打针的。

看到 “替布”（如托法替布）、“替尼”（如巴瑞替尼） -> 这是口服小分子靶向药（JAK抑制剂）。

看到 柳氮磺吡啶、来氟米特 -> 这是传统合成DMARDs。

【考点 c2.2.1｜M05｜必背要点】
必须死记（分类的“铁律”）：

治标不治本：NSAIDs（布洛芬）和激素，只能缓解症状，不能改变病情。

治本核心：甲氨蝶呤是传统改善病情药（DMARDs）的基石和首选。

新贵标签：依那西普、阿达木单抗是TNF抑制剂（生物制剂）；托法替布是JAK抑制剂（口服靶向药）。
理解记忆（做题的逻辑）：

用药顺序：先试传统药（如甲氨蝶呤），不行再上生物/靶向药。

名字玄机：名字带“单抗”的多半要注射；名字带“替布/替尼”的可以口服。

特殊成员：双醋瑞因、金诺芬也属于“治本”药，但相对小众，知道类别即可。

【考点 c2.2.1｜M06｜解题逻辑与秒杀规则】
解题黄金法则：先抓题干核心动词，它决定答案方向。
错误选项的经典长相（看到这种描述，大概率错）：

“布洛芬可以延缓类风湿关节炎的关节破坏。” -> 错（它只能止痛）。

“甲氨蝶呤是一种生物制剂。” -> 错（它是传统化学药）。

“轻中度类风湿关节炎，应首选依那西普治疗。” -> 错（首选一般是甲氨蝶呤等传统药）。

题干秒杀线索（条件反射）：

看到题干说 “迅速抗炎镇痛”、“对症治疗” -> 答案找NSAIDs（布洛芬） 或激素。

看到题干说 “控制病情活动”、“延缓影像学进展” -> 答案找改善病情药（DMARDs）。

看到选项中出现 “TNF-α抑制剂”、“IL-17抑制剂” -> 这描述的是生物制剂。

看到选项中出现 “JAK抑制剂” -> 这描述的是托法替布、巴瑞替尼这类口服药。

看到药名是 “XX单抗” -> 直接归类为生物制剂，排除“传统合成药”选项。

看到药名是 “XX替布”或“XX替尼” -> 直接归类为口服小分子靶向药。

第三阶段 冲刺秒杀 考前

【考点 c2.2.1｜M05｜必背要点】
考前最后3分钟，只背这“三个一”：

一个治标代表：布洛芬（NSAIDs）。

一个治本核心：甲氨蝶呤（传统DMARDs）。

一个新药标签：依那西普（生物制剂）、托法替布（靶向药）。

【考点 c2.2.1｜M06｜解题逻辑与秒杀规则】
看到以下关键词，直接进行条件反射，秒选答案：

看到 “镇痛消肿”、“对症” -> 秒想 布洛芬、双氯芬酸（NSAIDs）。

看到 “延缓关节破坏”、“改善病情” -> 秒想 甲氨蝶呤、柳氮磺吡啶（传统DMARDs）。

看到 “TNF-α”、“IL”、“单抗” 这些词 -> 秒想 依那西普、阿达木单抗（生物制剂）。

看到 “JAK”、“口服靶向” -> 秒想 托法替布、巴瑞替尼。

看到选项把 布洛芬 说成“改善病情” > 直接排除。

看到选项把 甲氨蝶呤 说成“生物制剂” > 直接排除。
