第一阶段 建立框架 初学

【考点 c2.2.2｜M02｜本页定位】
本页帮你用5分钟理清抗风湿“治本药”的核心：记住谁是老大，谁有特殊规矩，谁负责对付什么病。解决看见药名就发懵的问题。

【考点 c2.2.2｜M03｜考什么 & 怎么考】
考试主要考四件事：
第一，考谁是类风湿关节炎的首选基础药（锚定药）。
第二，考哪些药有独特且必须记住的用药规矩（比如剂量、监测、禁忌）。
第三，考哪个药能治疗特定的合并疾病（比如关节炎合并肠炎）。
第四，考通过不良反应或监测要求，反过来猜是哪个药。

【考点 c2.2.2｜M04｜核心结构】
抗风湿“治本药”分为两大阵营，要记牢他们的带头大哥和关键成员：
第一阵营：传统合成改善病情药（基础部队）

老大（锚定药）：甲氨蝶呤（类风湿首选）。

关键成员：来氟米特（一天一次，有特殊用法）、金诺芬（起效慢，要按月体检）、双醋瑞因（专攻骨关节炎，路线特殊）。
第二阵营：生物制剂和靶向药（特种部队）

关键成员：托法替布（口服靶向药）、英夫利西单抗（能管肠道）、司库奇尤单抗（对某些肠病要小心）。

【考点 c2.2.2｜M05｜必背要点】
现在只背这三条，建立最高频考点印象：

类风湿关节炎治疗，首选的“锚定药”和基础是甲氨蝶呤。

用来氟米特，前三天要“负荷剂量”每天50mg，之后每天10或20mg，一天一次。孕妇、哺乳妈妈、肝肾功能严重不好的人禁用。

用金诺芬，起效很慢（常需3个月），用药后必须至少每个月复查血、尿、肝肾功能。

第二阶段 复习查漏 默认推荐

【考点 c2.2.2｜M03｜考什么 & 怎么考】
命题人挖坑主要围绕以下几个“记忆薄弱点”：
坑一：动摇“老大”地位。选项说“类风湿首选不是甲氨蝶呤”，这就是错的。
坑二：在细节规矩上做文章。故意写错来氟米特的用法、金诺芬的监测频率。
坑三：用不良反应“钓鱼”。题干描述“间质性肺炎”、“叶酸缺乏”，就是等你选甲氨蝶呤。
坑四：在适应症上“张冠李戴”。把能治炎症性肠病的药安到别人头上，或忽略司库奇尤单抗对肠病的谨慎要求。
坑五：否认“特种部队”的风险。声称生物制剂或靶向药“绝对安全”、“不增加肿瘤风险”，这种绝对化描述往往是错的。

【考点 c2.2.2｜M04｜核心结构】
这是完整的“用药决策流程图”，做题时按步骤走：
第一步：判断疾病与阶段

如果是“类风湿关节炎”需要开始“治本” -> 优先考虑甲氨蝶呤。

如果明确说“甲氨蝶呤疗效不足的中重度类风湿” -> 考虑升级到托法替布等靶向药。
第二步：排查人群禁忌（优先于一切）

只要题干提到“妊娠”、“哺乳”、“严重肝/肾功能损伤” -> 立刻排除来氟米特。
第三步：识别“不良反应”或“监测”线索

题干出现“间质性肺炎”、“建议补充叶酸”、“肝功能异常/肝硬化” -> 高度指向甲氨蝶呤。

题干强调“起效慢（3个月）”、“需每月监测血尿常规、肝肾功能” -> 锁定金诺芬。
第四步：处理合并疾病

患者同时有“克罗恩病”或“溃疡性结肠炎” -> 优先考虑英夫利西单抗（它有这个适应症）。

患者有“银屑病或强直性脊柱炎”，但同时有“炎症性肠病”病史 -> 对司库奇尤单抗要持“慎用”态度。
第五步：注意特殊药物机制

治疗“骨关节炎”的药物，提到“不抑制前列腺素”、“通过抑制IL-1起作用” -> 说的是双醋瑞因。注意它怕和含铝、镁的抗酸药同服，会影响吸收。
第六步：牢记通用风险

TNF抑制剂（如英夫利西单抗）和JAK抑制剂（如托法替布）都有“诱发感染、肿瘤”的潜在风险，这是它们的共性标签。

【考点 c2.2.2｜M05｜必背要点】
必须死记（做题的钢印）：

甲氨蝶呤：类风湿锚定药。机制抑制二氢叶酸还原酶。关键不良反应：间质性肺炎、肝损伤、需补充叶酸。

来氟米特：用法特殊。前3天负荷剂量50mg/日，之后维持10或20mg/日，一日一次。禁忌：妊娠、哺乳、严重肝肾功能不全。

金诺芬：起效慢（3-6个月）。必须用药前检查，用药后至少每月复查血尿常规、肝肾功能。
理解记忆（用于区分和判断）：

双醋瑞因：用于骨关节炎，作用机制不同（抑制IL-1），避免与氢氧化铝/镁同服。

英夫利西单抗：除关节炎外，还可用于克罗恩病、溃疡性结肠炎。

司库奇尤单抗：用于银屑病、强直，但对有炎症性肠病患者需谨慎。

风险提示：所有生物制剂和靶向药（TNFi、JAKi）均需警惕感染和肿瘤风险。

【考点 c2.2.2｜M06｜解题逻辑与秒杀规则】
解题核心：像查户口一样，抓住题干的关键信息点，去匹配药物的唯一性特征。
错误选项的经典长相（看到就警惕）：

“类风湿关节炎应首选来氟米特而非甲氨蝶呤。”（错，动摇老大地位）

“来氟米特可用于妊娠期类风湿患者。”（错，触犯绝对禁忌）

“金诺芬起效迅速，无需特殊监测。”（错，违背核心特征）

“双醋瑞因通过抑制前列腺素合成治疗骨关节炎。”（错，机制弄错）

“TNF抑制剂非常安全，无肿瘤风险。”（错，否认共性风险）

题干关键词秒杀对应：

看到“类风湿关节炎”、“初始治疗”、“基础用药” -> 秒想甲氨蝶呤。

看到“间质性肺炎”、“补充叶酸” -> 秒想甲氨蝶呤。

看到“妊娠期/哺乳期禁用” -> 秒想来氟米特（并排除）。

看到“每月监测血尿常规” -> 秒想金诺芬。

看到“骨关节炎”、“IL-1抑制剂” -> 秒想双醋瑞因。

看到“合并克罗恩病/溃疡性结肠炎” -> 秒想英夫利西单抗。

看到“银屑病合并肠病史，用药需谨慎” -> 秒想司库奇尤单抗。

第三阶段 冲刺秒杀 考前

【考点 c2.2.2｜M05｜必背要点】
最后三分钟，只背这三组“唯一”对应，形成条件反射：

类风湿首选锚定药 -> 甲氨蝶呤。

妊娠哺乳绝对禁 + 前三天负荷量 -> 来氟米特。

起效慢+按月监测 -> 金诺芬。

【考点 c2.2.2｜M06｜解题逻辑与秒杀规则】
看到题干关键词，直接得出答案，不要犹豫：

看到“类风湿基础用药” -> 直接选“甲氨蝶呤”。

看到“间质性肺炎”或“需补叶酸” -> 直接联想“甲氨蝶呤”。

看到“妊娠期” -> 直接排除“来氟米特”。

看到“每月复查血尿常规” -> 直接选“金诺芬”。

看到“骨关节炎，不抑制前列腺素” -> 直接选“双醋瑞因”。

看到“关节炎合并克罗恩病” -> 直接选“英夫利西单抗”。

看到“TNF抑制剂” -> 直接想到“有感染和肿瘤风险”。
