第一阶段 建立框架 初学

【考点 c2.1.2｜M02｜本页定位】
本页帮你快速看懂NSAIDs怎么用才安全。重点就三件事：1. 什么情况选什么药（选对）。2. 哪些人不能用（避开）。3. 跟什么药一起吃会出事（防坑）。

【考点 c2.1.2｜M03｜考什么 & 怎么考】
考试主要从四大场景考你，都跟临床决策直接相关：
场景一：给一个具体病人（如儿童、胃溃疡者、冠心病者），让你选最合适或最不合适的药。
场景二：给出治疗目的（如退热、抗炎、预防血栓），让你判断用药选择是否正确。
场景三：考查某个药的“专属”不良反应或禁忌（如尼美舒利伤肝、塞来昔布心血管风险）。
场景四：判断NSAIDs与其他药物合用时，风险是增加还是减少。

【考点 c2.1.2｜M04｜核心结构】
临床用药评价，围绕“用途”和“风险”两大支柱来记。
用途轴（用来干什么）：

解热/镇痛：多数NSAIDs都行，但对乙酰氨基酚是“退热镇痛专业户”（几乎不抗炎）。

抗炎/抗风湿：需要抑制COX-2，对乙酰氨基酚不行。

抗血小板（预防血栓）：这是阿司匹林的“独家专利”。

风险轴（要防什么）：

伤胃（非选择性药风险高）

伤心（塞来昔布等COX-2选择性药风险高）

伤肾（所有NSAIDs都可能）

伤肝（对乙酰氨基酚过量、尼美舒利）

增加出血（尤其与抗凝药合用）

药物相互作用（与降压药、锂盐等）

【考点 c2.1.2｜M05｜必背要点】
现阶段只背通这三条“总纲”：

儿童发热镇痛，首选对乙酰氨基酚，禁用阿司匹林（防瑞夷综合征）。

对乙酰氨基酚几乎无抗炎作用，过量会伤肝；阿司匹林能抗血小板防血栓。

选药时记住风险口诀：非选择性药伤胃，选择性药（如塞来昔布）伤心，所有药都可能伤肾。

第二阶段 复习查漏 默认推荐

【考点 c2.1.2｜M03｜考什么 & 怎么考】
命题陷阱高度集中在以下五类“混淆”和“遗漏”，专考记不牢的人：
陷阱一：混淆药效。故意夸大对乙酰氨基酚的抗炎作用，或说布洛芬能抗血小板。
陷阱二：忽略人群禁忌。尤其在儿童（阿司匹林）、老年人（肝肾功能）、特定病史（胃溃疡、心梗）患者用药上设坑。
陷阱三：记反风险特征。说非选择性药（如布洛芬）心血管风险最高，或说塞来昔布对胃绝对安全。
陷阱四：遗漏独有毒性。不提及尼美舒利的肝毒性、吲哚美辛的严重不良反应。
陷阱五：忽视致命相互作用。忽略NSAIDs与抗凝药、ACEI类降压药、锂盐合用时的严重风险。

【考点 c2.1.2｜M04｜核心结构】
这是临床决策的完整地图，必须按此逻辑思考：
第一步：看人群与病史（先排除绝对不能用的）

儿童：发热/镇痛只选对乙酰氨基酚或布洛芬；禁用阿司匹林（瑞夷综合征）；禁用尼美舒利（12岁以下）。

有胃溃疡/出血史：慎用布洛芬、阿司匹林等非选择性药；可考虑塞来昔布（但需评估心脏）。

有冠心病、心梗、卒中史：禁用或慎用塞来昔布（增加血栓风险）。

严重肝病：慎用对乙酰氨基酚、尼美舒利。

脱水、老年人、肾功能不全：所有NSAIDs都需格外警惕，易致肾损伤。

第二步：看治疗目的（选最适合的）

仅需退热或轻度镇痛（如感冒头痛）：首选对乙酰氨基酚。安全窗较宽，但成人日剂量不超过2g。

需要抗炎（如关节炎、痛风）：选布洛芬、双氯芬酸、萘普生等；对乙酰氨基酚无效。

需要抗血小板（预防心梗、脑梗）：唯一选择小剂量阿司匹林。其他NSAIDs无此作用，且可能干扰阿司匹林效果。

第三步：看合并用药（防相互作用“地雷”）

与华法林、阿司匹林等抗凝/抗血小板药合用：出血风险剧增，应避免或严密监测。

与“普利”或“沙坦”类降压药（如卡托普利、氯沙坦）合用：不仅降压效果减弱，还可能引发急性肾衰竭，尤其对脱水或肾病患者风险极高。

与利尿剂、环孢素、锂盐合用：均会增加肾毒性或中毒风险。

阿司匹林与其他NSAIDs（如布洛芬）合用：不增加疗效，但胃出血风险显著增加。

【考点 c2.1.2｜M05｜必背要点】
必须死记（看到题干关键词就要反应）：

两个“首选一禁”：儿童退热首选对乙酰氨基酚/布洛芬，禁用阿司匹林。

两个“独有”：只有对乙酰氨基酚几乎不抗炎；只有阿司匹林用于抗血小板预防。

三个“风险指向”：胃不好怕非选择性药；心脏不好怕塞来昔布；肾不好怕所有NSAIDs。
理解记忆（掌握原因才能灵活判断）：

剂量是生命线：对乙酰氨基酚肝毒性与剂量直接相关，成人日极限2g。

合用即风险：NSAIDs与抗凝药、ACEI/ARB类降压药合用，风险不是叠加是倍增。

老人要减量：老年人肝肾功能下降，对NSAIDs的不良反应更敏感，必须从最小剂量开始。

【考点 c2.1.2｜M06｜解题逻辑与秒杀规则】
解题核心逻辑：像医生一样，遵循“安全>有效”原则。先排查禁忌人群和相互作用，再考虑疗效。
错误选项的经典长相：

“对乙酰氨基酚是强效抗炎药” -> 错（它不抗炎）。

“塞来昔布胃肠道安全性高，可完全放心用于所有胃溃疡患者” -> 错（有心血管风险，且非绝对安全）。

“为增强抗炎效果，可将阿司匹林与布洛芬联用” -> 错（增加胃出血风险，且可能对抗血小板作用产生干扰）。

“NSAIDs与ACEI类药物合用是安全的” -> 错（有急性肾衰风险）。

题干秒杀线索：

题干出现 “儿童”、“病毒感染后发热” -> 答案中找 “对乙酰氨基酚” ，排除 “阿司匹林”。

题干出现 “既往胃溃疡病史” -> 慎用/避免 “布洛芬、阿司匹林、双氯芬酸” 等。

题干出现 “心肌梗死病史”、“卒中史” -> 禁用 “塞来昔布”。

题干出现 “正在服用华法林” 或 “服用卡托普利降压” -> 高度警惕合用NSAIDs的风险，通常建议避免或需严密监测。

题干说 “长期服用” -> 特别注意 “对乙酰氨基酚的肝毒性” 和 “所有NSAIDs的肾毒性”。

第三阶段 冲刺秒杀 考前

【考点 c2.1.2｜M05｜必背要点】
最后冲刺，只背这四句，覆盖80%考点：

儿童发热：对乙酰氨基酚（可），阿司匹林（禁）。

抗炎：对乙酰氨基酚（不行），其他NSAIDs（行）。

抗血栓：阿司匹林（行），其他NSAIDs（不行）。

风险：非选择性伤胃，塞来昔布伤心，全都可能伤肾。

【考点 c2.1.2｜M06｜解题逻辑与秒杀规则】
看到以下关键词，直接进行条件反射，无需犹豫：

看到“儿童” -> 答案里找 “对乙酰氨基酚” 或 “布洛芬” ，但凡出现 “阿司匹林” 直接排除。

看到“仅需退热”、“轻度镇痛” -> 优先考虑 “对乙酰氨基酚”。

看到“类风湿关节炎”、“痛风急性发作”（需要抗炎） -> 排除 “对乙酰氨基酚”。

看到“预防心肌梗死” -> 锁定 “阿司匹林”。

看到“胃溃疡病史” -> 避开 “布洛芬、萘普生、双氯芬酸” 等非选择性药。

看到“冠心病史”、“心梗史” -> 禁用 “塞来昔布”。

看到“合用华法林” -> 答案通常是 “增加出血风险” 或 “避免合用”。

看到“合用卡托普利” -> 答案通常是 “影响降压，警惕肾损伤”。

看到选项说“两种NSAIDs联用效果更好” -> 直接判为错误选项。
