第一阶段：新兵报到——分清四类“肠道交警”
【考点 c4.2.3｜M02｜本页定位】
兄弟们，肠道不动了、堵车了怎么办？就得请出“肠道交警”——胃肠动力药。这帮交警分属不同“支队”，指挥交通的方式也完全不同。本页目标很简单：看到药名，立刻知道它是哪个支队的，用什么方式“疏通道路”。

【考点 c4.2.3｜M03｜考什么 & 怎么考】
关于分类，出题老师就爱玩“对号入座”：
一、给警号（药名），猜支队（类别）：给你一个药名（比如“莫沙必利”），问你它属于哪类动力药。
二、给指挥方式（机制），点名警员（代表药）：告诉你一种作用机制（比如“5-HT4受体激动剂”），让你从选项里挑出谁是它的成员。

【考点 c4.2.3｜M04｜核心结构】
“肠道交警大队”下设四个核心支队，指挥逻辑各异：

第一支队：全能纠察队（多巴胺D2受体拮抗剂）

指挥逻辑：他们是“纠察”，专门打击抑制胃肠蠕动的“坏分子”——多巴胺（D2受体）。把多巴胺按住，胃肠自然就有劲动了。

明星警员：甲氧氯普胺（胃复安）。这位是老牌，管得宽，既在中枢（大脑） 当纠察，也在外周（肠道） 当纠察。

第二支队：外围执勤队（外周多巴胺D2受体拮抗剂）

指挥逻辑：他们也是“纠察”，但只在外围（胃肠道）执勤，不进入大脑中枢。

明星警员：多潘立酮（吗丁啉）。口号是：“大脑的事我不管，我只管肠道畅通。”

第三支队：信号放大队（5-HT4受体激动剂）

指挥逻辑：他们是“信号兵”，通过激动5-HT4受体这个“对讲机”，放大“向前蠕动”的信号，促进神经末梢释放乙酰胆碱（动力燃料）。

明星警员：莫沙必利。这是新一代，选择性高，不干扰其他系统。

第四支队：双管齐下队（双重机制促动力药）

指挥逻辑：他们有两手：一手当“纠察”（拮抗D2受体），另一手当“加油工”（抑制乙酰胆碱酯酶，让已有的动力燃料“乙酰胆碱”少被分解）。

明星警员：伊托必利。双倍努力，效果更强。

【考点 c4.2.3｜M05｜必背要点】
新人报到，先记住三个支队的核心特征：

甲氧氯普胺：是全能纠察，中枢和外周都管。

多潘立酮：是外围执勤，只负责外周（肠道），不惹大脑。

莫沙必利：是信号兵，通过激动5-HT4受体来发号施令。

第二阶段：实战演练——识别警种，破解混淆
【考点 c4.2.3｜M03｜考什么 & 怎么考】
这里的陷阱，全是“指鹿为马”和“混淆职权”：

“支队混淆”坑：把属于“信号放大队”（莫沙必利）的成员，硬说成是“多巴胺受体纠察”（多潘立酮类），或者反过来。

“执勤范围”坑：模糊甲氧氯普胺和多潘立酮最大的区别——一个能进中枢，一个不能。题干里一提“中枢副作用”，考的就是这个。

“独有机制”坑：把伊托必利的双重机制，错误描述为“5-HT4受体激动”，或者把莫沙必利说成有“抗多巴胺”作用。

【考点 c4.2.3｜M04｜核心结构】
快速识别“交警”身份的四步口诀：
第一步：看它是不是“全能纠察”甲氧氯普胺？

特点：能进入大脑（有中枢作用），止吐效果好，但可能引起嗜睡、锥体外系反应。

题目提到 “中枢止吐” 或出现 “锥体外系反应” → 优先想到它。

第二步：看它是不是“外围执勤”多潘立酮？

特点：不进入大脑（无中枢作用），专促胃肠动力，中枢副作用少。

题目强调 “仅外周作用” 或 “避免中枢副作用” → 优先想到它。

第三步：看它是不是“信号兵”莫沙必利？

特点：通过激动 “5-HT4受体” 来起作用，跟多巴胺系统无关。

题目出现 “5-HT4受体激动” 或 “促乙酰胆碱释放” → 秒锁它。

第四步：看它是不是“双管齐下”伊托必利？

特点：既有 “拮抗D2受体”（纠察），又有 “抑制乙酰胆碱酯酶”（加油）。

题目描述 “双重作用机制” → 优先想到它。

核心易混点澄清（选择题必考）：

莫沙必利 ≠ 多巴胺受体拮抗剂。它是5-HT4受体激动剂，是另一条路。

伊托必利 ≠ 5-HT4受体激动剂。它是D2拮抗 + 乙酰胆碱酯酶抑制剂。

记清谁有中枢作用（甲氧氯普胺），谁没有（多潘立酮、莫沙必利、伊托必利）。

【考点 c4.2.3｜M05｜必背要点】
必须死记的“警员花名册”：

多巴胺D2受体拮抗剂（纠察队）代表：甲氧氯普胺、多潘立酮。

5-HT4受体激动剂（信号队）代表：莫沙必利。

需要理解的“执勤特点”：

多潘立酮因为不进入中枢，所以锥体外系反应等中枢副作用比甲氧氯普胺少。

伊托必利通过拮抗D2 + 抑制乙酰胆碱酯酶双重机制来增强动力，效果可能更全面。

【考点 c4.2.3｜M06｜解题逻辑与秒杀规则】
做题就是“查证件”：

题干提到 “中枢止吐” 或副作用出现 “锥体外系反应” → 直接选 甲氧氯普胺。

题干强调 “仅用于促胃肠动力，无中枢作用” → 直接选 多潘立酮。

题干出现 “5-HT4受体激动” 或 “选择性促乙酰胆碱释放” → 直接选 莫沙必利。

题干描述 “兼具多巴胺D2拮抗和乙酰胆碱酯酶抑制” → 直接选 伊托必利。

看到选项把 “莫沙必利” 的作用机制写成拮抗多巴胺受体 → 那是错误选项。

警惕！这些是经典的“伪造身份”选项：

“莫沙必利为多巴胺D2受体拮抗剂。”

“伊托必利通过激动5-HT4受体发挥促动力作用。”

混淆甲氧氯普胺和多潘立酮的中枢作用有无。

第三阶段：考前冲刺——对号入座，秒选答案
【考点 c4.2.3｜M05｜必背要点】
上考场前，心里默念三组对应关系：

甲氧氯普胺 → 中枢+外周，全能纠察。

多潘立酮 → 仅外周，外围执勤。

莫沙必利 → 5-HT4激动，信号兵。

【考点 c4.2.3｜M06｜解题逻辑与秒杀规则】
形成条件反射，直接连线：

看到 “中枢作用/止吐/锥体外系反应” → 答案：甲氧氯普胺。

看到 “仅外周作用/无中枢副作用” → 答案：多潘立酮。

看到 “5-HT4受体激动” → 答案：莫沙必利。

看到 “双重机制/抑制乙酰胆碱酯酶” → 答案：伊托必利。
